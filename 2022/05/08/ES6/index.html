# Symbol 类型

ES6 引入了新的原始数据 Symbol 表示独一无二的值。它是 javascript 的第七种数据类型，是一种类似于字符串的数据类型

  1. Symbol 的值是 唯一 的，是用来解决命名冲突的问题
  2. Symbol 的值 不能与其他数据进行运算
  3. Symbol 定义的对象属性 不能使用for in 循环遍历 ，但是可以使用 Reflect.ownKeys 来获取对象的所有键名

使用场景：给对象添加属性和方法（表示独一无二）

# 迭代器 iterator

迭代器是一个接口 Itertor, 为各种不同的数据结构提供统一的访问机制，任何数据结构只要不输 Iterator, 就可以完成遍历操作。

  • ES6 创造了一种新的遍历命令 for...of 循环，Iterator 接口主要供 for...of 消费

  • 原生具备 iterator 接口的数据（可用 for...of 遍历）

工作原理

  • 创造一个指针对象，指向当前数据结构的位置
  • 第一次调用对象的 next 方法，指针自动指向数据结构的第一个成员
  • 接下来不断调用 next 方法，指针一直完后移动，直到指向最后一个成员
  • 每调用 next 方法返回一个包含 value 和 done 属性的对象

# Promise

Promise 是 ES6 引入的异步编程的新解决方案。语法上 Promise 是一个构造函数，用来封装异步操作可以获取其成功或者失败的结果

# Set 集合

ES6 提供了新的数据结构 Set (集合), 它 类似于数组 ， 但成员的值都是唯一的 ， 实现自动去重 。集合 实现了iterator接口 ，所以可以使用扩展运算符和 for of 进行遍历，集合的属性和方法。

  • size 返回集合的元素个数
  • add 增加一个新元素
  • delete 删除元素，返回 boolean 值
  • has 检测集合中是否包含某个元素，返回 boolean 值
  • clear 清空集合

应用：

  1. 数组去重
  2. 数组交集
  3. 数组并集

# Map

ES6 提供了 Map 数据结构，它 类似于对象 。也是键值对的集合。但是键的范围不限于字符串，各种类型的值（包括对象）都可以当作键，Map 也实现了iterator接口 ，所以可以使用扩展运算符和 for of 进行遍历，Map 的属性和方法

  • size 返回 Map 元素个数
  • set 增加新元素，返回当前 Map
  • get 赶回键名对象的键值
  • has 返回 Map 是否包含某个元素，返回 boolean 值
  • clear 清空集合，返回 undefined
