# 定义

TypeScript，通俗来说就是 JavaScript 的增强版。让所有的变量和属性有一个具体类型。

好处

  1. 开发过程中，发现潜在问题
  2. 更友好的编辑器自动提示

demo.ts

# 环境基础

  • 安装 Node.js 官网看安装教程
  • 安装 typescript npm i typescript -g
  • 安装 ts-mode npm i ts-node -g

复杂运行：ts 文件不能直接运行。运行 typescript 文件，先把 ts 文件装换成 js 文件，才能运行

  1. 生为 js 文件命令 tsc demo.ts
  2. 运行 js 文件 node demo.js

简易运行：安装 ts-node 直接运行

  1. 安装 ts-node npm i ts-node -g
  2. 运行 ts 文件 ts-node demo.ts
