# ES6 模块化规范

** 含义：** 浏览器和服务器端通用的模块化开发规范。他的出现极大的降低了前端开发者的模块化学习成本，开发者不需额外学习 AMD CMD 或者 CommonJS 等模块化规范。

ES6 模块化规范中定义：

  • 每个 js 文件都是独立的模块

  • 导入其它模块成员都是使用 import 关键字

  • 向外共享模块化成员都是 export 关键字

# node.js 中体验 ES6 模块化

mode.js 中默认仅支持 Common.js 模块化规范，若想基于 node.js 体验 ES6 的模块化语法，可以按照如下步骤进行：

  • 安装 v14.15.1 或更高版本的 node.js 终端 node -v 查看版本
  • 在 package.json 的根节点中添加 “type”:“module” 节点
