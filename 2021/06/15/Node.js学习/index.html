# NPM

npm: 包管理工具

全局安装： npm install 后面再加上 -g

全局安装包的目录

  • Mac

    /Users/felix/.nvm/versions/node/nvm各个版本/bin/

  • Windows

    C:\Users\你的用户名\AppData\Roaming\npm\node_modules

局部安装： 在当前目录 npm install

package.json 中设置脚本

命令

描述各种包以树的方式 npm list

查看 gulp 包关系： npm list | grep gulp

开发依赖: -D // 开发使用

生产环境包在 package.json 中的 devDependencies 属性中

生产依赖： -S // 线上使用

生产环境包在 package.json 中的 dependencies 属性中

node module 不需要复制要忽略提交，本地需要通过 npm 下载即可

全局安装 npm i

仅安装生产依赖 npm i --production

仅安装开发依赖 npm i --development

安装指定版本的包

$ pwd
$ npm list
$ npm info underscore
$ npm view underscore versions
$ npm install underscore@1.8.0
$ npm list
$ npm uninstall underscore
$ npm list
